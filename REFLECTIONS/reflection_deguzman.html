Joshua De Guzman (Reflection)

Working on the Poocafe case study taught me more than just writing code it taught me patience, perseverance, and how to survive on caffeine and last-minute fixes. I learned how to create a responsive coffee-themed website using HTML and CSS, and how to make things look good without crying over keyframes and flexbox (okay, maybe just a little). It was also a great experience seeing how all the little parts layout, colors, fonts come together like the perfect cup of coffee.

One major challenge was getting the layout to behave on different screen sizes. At one point, the navbar looked like it had a life of its own on mobile. But with some teamwork, Googling, and a few “why is this not working?!” moments, we fixed it. Managing the CSS was also chaotic at first it was like spaghetti code, but we cleaned it up by splitting it into separate files like responsible developers (or at least developers trying their best).

If I had more time and experience, I probably would’ve added some JavaScript magic to make the site more interactive maybe a dark mode switch, or an animated menu that slides in like a ninja. I also would’ve focused more on accessibility, like using proper semantic tags so screen readers wouldn’t hate me. And of course, I would’ve spent less time fixing typos and more time polishing the design to make it look more professional.

I want to dive deeper into JavaScript and frameworks like React so I can build cooler, more dynamic websites. I also want to learn more about UI/UX design, because making something look nice and work smoothly is a double win. Plus, getting better at PHP and SQL would help me become a full-stack wizard, capable of summoning both front-end beauty and back-end brains.

Overall, Poocafe was a wild but fun ride. I got to level up my skills, make mistakes, fix them, and still have a good time. It reminded me that learning to code is like brewing coffee you might mess up the first few times, but eventually, you get something worth sipping on.

This project taught me that coding is basically controlled chaos like trying to make a latte while riding a unicycle. There were bugs, broken layouts, and moments where I questioned my life choices but somehow, it all came together. In the end, Poocafe turned out to be more than just a website it was a digital rollercoaster fueled by coffee, confusion, and the occasional Google search that saved my life.
